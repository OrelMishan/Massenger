נכנס לתקיה שאליה נרצה לעשות clone
נריץ את הפקודה git clone https://github.com/OrelMishan/Massenger.git
נכנס לתקיה עם כל הקבצים (Messanger)
ונריץ את הפקודה npm install
(אם נכשל נמחק את הקובץ package-lock.json מהתקיה ונריץ npm install --force)
נריץ npm start
וכעת בכתובת localhost:3000 תעלה האפליקציה שלנו
